Step-by-step Setup Guide

🚀 Enhanced Version Benefits:
• Process unlimited files (no more 5-file limit)
• Advanced queue management with automatic retry
• Real-time progress tracking with WebSocket updates
• Smart rate limiting to prevent API quota issues

To access your BIM 360 Docs files with this enhanced app, you need to connect your BIM 360 account with the app.

  1. Log in to BIM 360 Account Administration . Note that you need account administrator permissions.
  2. If you have more than one account, select the account you want to integrate with the app.
  3. From the toolbar, select SETTINGS > Apps & Integrations.
  4. Click the Add Integration button.
  5. Select BIM 360 Account Administration and BIM 360 Docs, and click Next.
  6. Select I'm the developer, and click Next.
  7. In the APS Client ID and App Name fields, enter the APS client ID and app name provided by the app.
  8. Select the I have saved the Account ID information securely checkbox.
  9. Click Save.

    10. 🎉 Congratulations! You can now access your BIM 360 Docs files and process unlimited files with the enhanced bulk processing feature.
